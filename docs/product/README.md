# Product Documents

This folder owns GaemiGuard's product direction documents.

Use these documents when a change affects product identity, user workflows,
agent scope, terminal/data surfaces, automation boundaries, or public-facing
positioning.

## Documents

| Document | Purpose |
| --- | --- |
| `docs/product/agent-first-direction.md` | Current product direction: personal investment agent first, with investment guard and local terminal surfaces as supporting pieces. |

## Maintenance

- Keep product direction aligned with `README.md`.
- Keep stage implications aligned with `docs/development-status.md` and `docs/stages/`.
- Keep architecture implications aligned with `docs/architecture/agent-runtime.md` and `docs/waterfall/03-agent-orchestration.md`.
- Update `docs/README.md`, `docs/agent-index.md`, and `docs/architecture/design-index.md` when adding a new core product document.
